# plus-ui-shadcn

基础目录结构
```
src/
  ├── assets/           # 静态资源
  ├── components/       # 组件
  │   └── ui/          # shadcn-vue组件
  ├── composables/      # 组合式函数
  ├── layouts/          # 布局组件
  ├── pages/           # 页面
  ├── router/          # 路由配置
  ├── stores/          # 状态管理
  ├── styles/          # 样式文件
  ├── types/           # 类型定义
  └── utils/           # 工具函数
```
